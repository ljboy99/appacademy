HTML Tags Cheat Sheet

Headings

Headings from 'h1' to 'h6'.
Every page should only have one 'h1' tag.

Paragraphs

Paragraps of text go in the 'p' tags.

Lists

Unordered lists are marked with bullet points and go in the 'ul' tag.

Types of lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered Lists

You can make an ordered list with the 'ol' tag.
Each item will use an 'li' tag.

Anchors (AKA Links)

The 'a' tag is used to create a link to another page.
It uses the href attribute to point to that links destination.

Apparently it is called an anchor because web designers use it to 'anchor' a URL to text
